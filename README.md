# 📈 Market Sentiment Intelligence Terminal
**Automated Live News Scraping | NLP Sentiment Analysis | Power BI Interactive Dashboard**

## 🌟 Project Overview
This project is an end-to-end data intelligence solution designed to track the "Market Mood" of the tech industry. It automates the collection of headlines from major tech sources and quantifies them using Natural Language Processing (NLP) to provide real-time sentiment signals.

## 🛠️ The Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas, NLTK (VADER Sentiment), Requests
* **Visualization:** Power BI Desktop
* **Data Modeling:** DAX (Custom Measures, Sentiment Logic), Sync Slicers

## 🚀 Key Features
* **Automated Data Pipeline:** Python script extracts and processes headlines into a structured dataset.
* **Sentiment Gauging:** Uses NLP to score headlines from -1 (Extremely Negative) to +1 (Extremely Positive).
* **Cross-Page Insights:** A "Command Center" for high-level trends and a "Diagnostic Lab" for headline-level auditing.
* **Smart UI:** Search-synced pages and heatmapped source analysis.

## 📊 How to Run
1. **Python Script:** Ensure you have the libraries installed (`pip install -r requirements.txt`) and run `sentiment.py`.
2. **Dashboard:** Open `Market_Sentiment_Dashboard.pbix` to view the live visualizations.
